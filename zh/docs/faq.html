常见问题

关于 Wegent 的常见问题解答。

通用问题​

什么是 Wegent？​

Wegent 是一个 AI 驱动的多 Agent 协作平台，使您能够创建、配置和编排 AI Agent 来执行各种任务。

系统要求是什么？​

  • Python 3.8 或更高版本
  • 最低 4GB RAM（建议 8GB）
  • 云 AI 提供商需要网络连接

Wegent 免费使用吗？​

Wegent 是开源且免费使用的。但是，根据您的使用情况，您可能需要支付 AI 模型提供商（OpenAI、Anthropic 等）的费用。

Agents​

Ghost 和 Bot 有什么区别？​

  • Ghost 是在后台工作的自主 Agent
  • Bot 是响应用户输入的交互式 Agent

我可以创建多少个 Agent？​

Agent 数量没有硬性限制。实际限制取决于您的系统资源和 API 速率限制。

Agent 之间可以通信吗？​

可以，Agent 可以通过团队进行通信，团队支持多种协作模式。

模型​

支持哪些 AI 模型？​

Wegent 支持：

  • OpenAI (GPT-4, GPT-3.5)
  • Anthropic (Claude)
  • 本地模型 (Ollama, LM Studio)
  • 通过 API 的自定义提供商

可以使用本地模型吗？​

可以，Wegent 通过 Ollama 和 LM Studio 集成支持本地模型。

故障排除​

为什么我的 Agent 没有响应？​

检查：

  1. API 密钥配置
  2. 模型可用性
  3. 网络连接
  4. 速率限制
